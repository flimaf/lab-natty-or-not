# IA para Iniciantes: Um Guia Rápido

## 📒 Descrição
Um e-book curto e visual que explica o que são Inteligências Artificiais Generativas, como funcionam e onde podemos usá-las no dia a dia. Feito especialmente para quem não tem conhecimento técnico.

## 🤖 Tecnologias Utilizadas
- **ChatGPT** para criar o texto explicativo.
- **Canva** para montar a capa e o layout do e-book.
- **Magic Media (IA do Canva)** para gerar a imagem de fundo da capa.

## 🧐 Processo de Criação
1. Usei o ChatGPT com o prompt: *"Crie um roteiro para um mini e-book de 3 páginas sobre IA para iniciantes, com linguagem simples e exemplos práticos."*
2. No Canva, escolhi um modelo de e-book gratuito e colei o texto gerado.
3. Para a capa, usei o Magic Media com o comando *"fundo abstrato tecnológico azul e roxo"* e adicionei o título com uma fonte moderna.
4. Exportei tudo como PDF e tirei um print da capa.
